## Setup Commands

After saving the file as `joe-compressor.sh`, run the following commands on your terminal:
```
mv joe-compressor.sh /usr/local/bin/jcmprs
chmod 4777 /usr/local/bin/jcmprs
```

---

## How to Use It

To compress a file:
```
jcmprs -c filename
```

To extract a file:
```
jcmprs -x filename
```
